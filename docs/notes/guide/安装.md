# 数据库

:::tip
请确保安装了docker和docker-compose
redis和MongoDb可以不使用docker安装，自行按照设置账号密码就行
:::

## redis

```
git clone https://github.com/Autumn-27/ScopeSentry-Scan
cd ScopeSentry-Scan/build
docker-compos -f redis-compose.yml up -d
```
